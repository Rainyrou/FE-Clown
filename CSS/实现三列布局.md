###### 1. Flexbox

```HTML
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Container</title>
    <style>
        #container {
            display: flex;
        }

        .left,
        .right {
            width: 20%;
            background-color: red;
        }

        .center {
            flex-grow: 1;
            background-color: blue;
        }
    </style>
</head>
<body>
    <div id="container">
        <div class="left">left</div>
        <div class="center">center</div>
        <div class="right">right</div>
    </div>
</body>
</html>
```

###### 3. Float

```HTML
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Container</title>
    <style>
        .left,
        .right {
	        float: left;
            width: 20%;
            background-color: red;
        }
        .center {
	        float: left;
            width: 60%;
            background-color: blue;
        }
    </style>
</head>
<body>
    <div id="container">
        <div class="left">left</div>
        <div class="center">center</div>
        <div class="right">right</div>
    </div>
</body>
</html>
```

块级元素在标准文档流中垂直排列，使用浮动后变为水平排列

###### 4. positioning

```CSS
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Container</title>
    <style>
        #container {
            position: relative;
        }

        .left,
        .right {
            position: absolute;
            top: 0;
            bottom: 0;
            width: 20%;
            background-color: red;
        }

        .left {
            left: 0;
        }

        .right {
            right: 0;
        }

        .center {
            margin-left: 20%;
            margin-right: 20%;
            background-color: blue;
        }
    </style>
</head>
<body>
    <div id="container">
        <div class="left">left</div>
        <div class="center">center</div>
        <div class="right">right</div>
    </div>
</body>
</html>
```
