实现多行文本省略即在文本溢出容器时显示省略号主要依赖 `-webkit-line-clamp` 属性，它用于限制在一个块元素显示的文本的行数，当文本超过设定行数时，多余的文本会被省略标记替换

```HTML
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ellipsis</title>
    <style>
        .ellipsis {
            display: -webkit-box;
            -webkit-box-orient: vertical;
            -webkit-line-clamp: 3;
            overflow: hidden;
            text-overflow: ellipsis;
            max-width: 300px;
            line-height: 1.2em;
            height: 3.6em;
        }
    </style>
</head>
<body>
    <div class="ellipsis">
        Here are some texts that may be very long, very long, very long, very long, very long, very long, very long,
        very long, so that we need to use the text omission effect to deal with it and keep the interface clean.
    </div>
</body>
</html>
```
