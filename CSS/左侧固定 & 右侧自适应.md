###### 1. Flexbox

`index.html`

```HTML
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <div id="container">
        <div class="left">left</div>
        <div class="right">right</div>
    </div>
    <link rel="stylesheet" href="style.css">
</body>
</html>
```

`style.css`

```CSS
#container {
    display: flex;
}

.left {
    width: 200px;
    flex-shrink: 0;
}

.right {
    flex-grow: 1;
}
```

###### 2. float

`style.css`

```CSS
.left {
    float: left;
    width: 200px;
}

.right {
    margin-left: 200px;
}
```
