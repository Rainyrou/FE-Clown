页面水印：

```css
.item {
  color: rgba(0, 0, 0, 0.1);
  font-size: 24px;
  font-weight: bold;
  text-align: center;
  transform: rotate(-30deg);
  user-select: none;
}
```

毛玻璃：

```css
.item {
  backdrop-filter: blur(10px);
  background: rgba(255, 255, 255, 0.1);
}
```

图片自适应：

```css
.item {
  width: 100%;
  height: 100%;
  object-fit: cover;
}
```

吸顶导航栏：

```css
.item {
  position: sticky;
  top: 0;
  background-color: blue;
  padding: 10px 0;
  z-index: 1000;
}
```
