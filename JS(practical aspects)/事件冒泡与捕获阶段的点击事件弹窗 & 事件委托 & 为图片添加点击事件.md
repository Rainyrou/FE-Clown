事件冒泡与捕获阶段的点击事件弹窗：

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    .box {
      padding: 20px;
      border: 2px solid black;
      margin: 10px;
    }
  </style>
</head>
<body>
  <div id="outer" class="box">Outer
    <div id="middle" class="box">Middle
      <div id="inner" class="box">Inner</div>
    </div>
  </div>
  <script>
    const showPopup = (message) => alert(message);
    
    document
      .getElementById("outer")
      .addEventListener("click", (e) => showPopup("捕获阶段：outer"), true);

    document
      .getElementById("middle")
      .addEventListener("click", (e) => showPopup("捕获阶段：middle"), true);

    document
      .getElementById("inner")
      .addEventListener("click", (e) => showPopup("捕获阶段：inner"), true);

    document
      .getElementById("outer")
      .addEventListener("click", (e) => showPopup("冒泡阶段：outer"), false);

    document
      .getElementById("middle")
      .addEventListener("click", (e) => showPopup("冒泡阶段：middle"), false);

    document
      .getElementById("inner")
      .addEventListener("click", (e) => showPopup("冒泡阶段：inner"), false);
  </script>
</body>
</html>
```

事件委托：

```JavaScript
function _addEventListener(element, type, fn, selector) {
  if (!selector) element.addEventListener(type, fn);
  else {
    element.addEventListener(type, function (event) {
      const target = event.target;
      if (target.matches(selector)) fn.call(target, event);
    });
  }
}
```

请补全 JavaScript 代码，要求如下：

1. 给"ul"标签添加点击事件
2. 当点击某"li"标签时，该标签内容拼接"."符号。如：某"li"标签被点击时，该标签内容为".."

注意：必须使用 DOM0 级标准事件（onclick）

```js
document.querySelector("ul").onclick = (event) => {
  const target = event.target;
  if (target.tagName.toLowerCase() === "li") target.innerHTML += ".";
};
```
