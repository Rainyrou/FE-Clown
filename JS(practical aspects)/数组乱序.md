2. Fisher-Yates/Knuth 洗牌算法：

```JavaScript
const shuffle = (arr) => {
  const len = arr.length;
  for (let i = len - 1; i > 0; --i) {
    const j = Math.floor(Math.random() * (i + 1));
    [arr[i], arr[j]] = [arr[j], arr[i]];
  }
  return arr;
};
```

时间复杂度 O(n)
