
| 占位符       | 方法            | 说明                | 逻辑                          |
| --------- | ------------- | ----------------- | --------------------------- |
| `yyyy/yy` | `getFullYear` | 返回四位数年份（如 2024）   | `yy` 通过 `%100` 取后两位         |
| `MM/M`    | `getMonth`    | 返回月份（0-11，0 = 一月） | 加 1 对应实际月份，`MM` 补零          |
| `dd/d`    | `getDate`     | 返回当月日期（1-31）      | `dd` 补零                     |
| `HH/H`    | `getHours`    | 返回 24 小时制小时（0-23） | `HH` 补零                     |
| `hh/h`    | `getHours`    | 返回 24 小时制小时（0-23） | 通过 `%12` 转换为 12 小时制，`hh` 补零 |
| `mm/m`    | `getMinutes`  | 返回分钟（0-59）        | `mm` 补零                     |
| `ss/s`    | `getSeconds`  | 返回秒（0-59）         | `ss` 补零                     |
| `w`       | `getDay`      | 返回星期（0-6，0 = 周日）  | 通过数组映射为中文星期几                |

日期格式化：

```JavaScript
const formatDate = (date, format) => {
  const addZero = (data) => (data < 10 ? "0" + data : data);
  const map = {
    yyyy: date.getFullYear(),
    yy: date.getFullYear() % 100,
    MM: addZero(date.getMonth() + 1),
    M: date.getMonth() + 1,
    dd: addZero(date.getDate()),
    d: date.getDate(),
    HH: addZero(date.getHours()),
    H: date.getHours(),
    hh: addZero(date.getHours() % 12),
    h: date.getHours() % 12,
    mm: addZero(date.getMinutes()),
    m: date.getMinutes(),
    ss: addZero(date.getSeconds()),
    s: date.getSeconds(),
    w: () => {
      const weekdays = ["日", "一", "二", "三", "四", "五", "六"];
      return weekdays[date.getDay()];
    },
  };
  for (const key in map) format = format.replace(key, map[key]);
  return format;
};

console.log(formatDate(new Date("2024-05-17 14:30:45"), "yyyy-MM-dd hh:mm:ss"));
```

一年第几天：

```js
const getDayOfYear = (dateString) => {
  const year = dateString.slice(0, 4);
  const month = dateString.slice(4, 6) - 1;
  const day = dateString.slice(6, 8);
  const date = new Date(year, month, day);
  const start = new Date(date.getFullYear(), 0, 0); // 前一年的最后一天
  const diff = date - start;
  const oneDay = 1000 * 60 * 60 * 24;
  const dayOfYear = Math.floor(diff / oneDay);
  return dayOfYear;
}
```

判断区间内每日步数是否达标：

```js
const check = (data, start, end, total) => {
  const startDate = new Date(start),
    endDate = new Date(end);
  const filtered = data.filter((item) => {
    const itemDate = new Date(item.date);
    return itemDate >= startDate && itemDate <= endDate;
  });
  return filtered.every((item) => item.step >= total);
};

const testData = [
  { date: "2025-03-20", step: 8000 },
  { date: "2025-03-21", step: 10000 },
  { date: "2025-03-22", step: 6000 },
];

console.log(check(testData, "2025-03-20", "2025-03-22", 7000)); // false
```
