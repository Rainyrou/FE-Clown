[1047. 删除字符串中的所有相邻重复项](https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/description/)

```javascript
var removeDuplicates = function (s) {
  const stk = [];
  for (const ch of s) {
    if (stk.length && stk[stk.length - 1] === ch) stk.pop();
    else stk.push(ch);
  }
  return stk.join("");
};
```
