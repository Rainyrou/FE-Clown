[125. 验证回文串](https://leetcode.cn/problems/valid-palindrome/description/)

```javascript
var isPalindrome = function (s) {
  s = s.replace(/[^a-zA-Z0-9]/g, "").toLowerCase();
  let left = 0,
    right = s.length - 1;
  while (left <= right) {
    if (s[left] !== s[right]) return false;
    ++left;
    --right;
  }
  return true;
};
```

`^` 于字符集开头表示取反
