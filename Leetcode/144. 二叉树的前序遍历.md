[144. 二叉树的前序遍历](https://leetcode.cn/problems/binary-tree-preorder-traversal/)

递归：

```javascript
var preorderTraversal = function (root) {
  if (!root) return [];
  const ans = [];
  const preOrder = (node) => {
    if (!node) return;
    ans.push(node.val);
    preOrder(node.left);
    preOrder(node.right);
  };
  preOrder(root);
  return ans;
};
```

迭代：

```javascript
var preorderTraversal = function (root) {
  const ans = [],
    stk = [];
  while (stk.length || root) {
    while (root) {
      ans.push(root.val);
      stk.push(root.right);
      root = root.left;
    }
    root = stk.pop();
  }
  return ans;
};
```

| 遍历类型 | 核心逻辑差异    | 处理根节点时机 | 子树压栈顺序      |
| ---- | --------- | ------- | ----------- |
| 中序遍历 | 左 → 根 → 右 | 弹出栈时处理  | 只压根节点，不压子树  |
| 前序遍历 | 根 → 左 → 右 | 压栈前处理   | 先压右子树，再移左子树 |
