[153. 寻找旋转排序数组中的最小值](https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array/description/?envType=study-plan-v2&envId=top-100-liked)

```javascript
var findMin = function (nums) {
  let left = 0,
    right = nums.length - 1,
    ans = Infinity;
  while (left <= right) {
    const mid = Math.floor((right - left) / 2) + left;
    if (nums[left] <= nums[mid]) {
      ans = Math.min(ans, nums[left]);
      left = mid + 1;
    } else {
      ans = Math.min(ans, nums[mid]);
      right = mid - 1;
    }
  }
  return ans;
};
```

原数组严格升序且元素互不相同，经过 1~n 次旋转后，数组变为两段严格升序的子数组，且满足：

* 左段子数组的所有元素 > 右段子数组的所有元素，因为旋转是将尾部元素移至头部，左段为原数组的后半部分，右段为原数组的前半部分
* 最小元素为两段子数组的分界点即右段子数组的首元素
* 整个数组仅有一个断点且左段末尾 > 右段开头，其余部分严格升序
* 当 `nums[left] <= nums[mid]` 说明其位于左段子数组，最小元素位于右段，排除左段
* 当 `nums[left] > nums[mid]` 说明其存在断点，最小元素位于其中，排除右段
