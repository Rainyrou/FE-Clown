[179. 最大数](https://leetcode.cn/problems/largest-number/description/)

```javascript
var largestNumber = function (nums) {
  nums.sort((a, b) => `${b}${a}` - `${a}${b}`);
  return nums[0] ? nums.join("") : "0";
};
```

`sort(a, b)`  若返回正数，则 b 前 a 后（交换 ab 位置），若返回负数，则 a 前 b 后，若返回 0，则位置不变

**哪个数在前拼接结果更大则排在前面**
