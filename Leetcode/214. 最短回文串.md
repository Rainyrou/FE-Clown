[214. 最短回文串](https://leetcode.cn/problems/shortest-palindrome/description/)

```
s[0..i-1] 为回文字符串 
↓ （由定义有） 
s[0..i-1] = 反转(s[0..i-1]) 
↓ （等价于） 
s[0..i-1] = reversed[len-i..len-1] 
```

```javascript
var shortestPalindrome = function (s) {
  const len = s.length;
  const reversed = s.split("").reverse().join("");
  for (let i = len; i >= 0; --i) {
    if (s.slice(0, i) === reversed.slice(len - i))
      return reversed.slice(0, len - i) + s;
  }
};
```
