[括号生成](https://leetcode.cn/problems/generate-parentheses/description/)

```javascript
var generateParenthesis = function(n) {
  const ans = [];
  const backTracking = (lRemain, rRemain, str) => {
    if(str.length === 2 * n) {
      ans.push(str);
      return;
    }
    if(lRemain) backTracking(lRemain - 1, rRemain, str + '(');
    if(lRemain < rRemain) backTracking(lRemain, rRemain - 1, str + ')');
  }
  backTracking(n, n, '');
  return ans;
};
```

* 有效括号满足左右括号一一匹配且任意子串前缀中的左括号数量 ≥ 右括号数量
* 剪枝：左括号的添加无前置条件 + 右括号的添加需满足剩余左括号数量 < 剩余右括号数量
* 隐式回溯：在 JavaScript 中字符串为不可变数据类型，每次字符串操作 `str + '('` 或 `str + ')'` 均创建新的字符串对象，每次递归均以修改后的字符串为参数，因此互不干扰，无需在递归返回后手动撤销修改
* 有效组合的构建过程必然为「先添加左括号，且右括号不早于左括号」—— 两个剪枝条件覆盖所有可能的添加路径
