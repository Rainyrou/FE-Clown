[328. 奇偶链表](https://leetcode.cn/problems/odd-even-linked-list/description/)

- while 循环终止条件为不存在下一个奇数节点
- 在最后一次 while 循环中 `even.next = odd.next === null` 完全没问题（链表节点的 `next` 可指向 `null`）

```javascript
var oddEvenList = function (head) {
  if (!head || !head.next) return head;
  let odd = head,
    even = head.next,
    evenHead = head.next;
  while (even && even.next) {
    odd.next = even.next;
    odd = odd.next;
    even.next = odd.next;
    even = even.next;
  }
  odd.next = evenHead;
  return head;
};
```
