[40. 组合总和 II](https://leetcode.cn/problems/combination-sum-ii/)

01 背包：外层 for 循环遍历物品，内层 for 循环遍历背包且倒序遍历

动态规划：23 / 176 个通过的测试用例（内存溢出）

```javascript
var combinationSum2 = function (candidates, target) {
  candidates.sort((a, b) => a - b);
  const dp = new Array(target + 1).fill(0).map(() => []);
  dp[0] = [[]];
  for (let i = 0; i < candidates.length; ++i) {
    for (let j = target; j >= candidates[i]; --j) {
      for (const item of dp[j - candidates[i]])
        dp[j].push([...item, candidates[i]]);
    }
  }
  const set = new Set(),
    ans = [];
  for (const item of dp[target]) {
    const key = item.join(",");
    if (!set.has(key)) {
      ans.push(item);
      set.add(key);
    }
  }
  return ans;
};
```
