[400. 第 N 位数字](https://leetcode.cn/problems/nth-digit/description/)

位数 → 目标数字 → 目标位

- 1 位数（1-9）：9 个数字，共 9×1=9 位
- 2 位数（10-99）：90 个数字，共 90×2=180 位
- 3 位数（100-999）：900 个数字，共 900×3=2700 位
- k 位数：9×10^(k-1)  个数字，共 9×10^(k-1)×k 位
- k 位数的特点为每个数字占用 k 个连续位置（如 2 位数 10 占用第 1、2 位，11 占用第 3、4 位，12 占用第 5、6 位...）
* 相对位置 n 映射至对应的 k 位数字，n 为 1-based，而数组索引为 0-based
- 目标位索引 = (n-1) % k（0-based 索引）

```javascript
var findNthDigit = function (n) {
  let k = 1; // 当前数字的位数
  let count = 9; // 当前位数的数字总数
  let start = 1; // 当前位数的起始数字
  while (n > k * count) {
    n -= k * count; // n 为相对位置
    ++k;
    count *= 10;
    start *= 10;
  }
  const target = start + Math.floor((n - 1) / k);
  const index = (n - 1) % k;
  return Number(String(target)[index]);
};
```
