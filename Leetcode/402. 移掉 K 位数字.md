[402. 移掉 K 位数字](https://leetcode.cn/problems/remove-k-digits/description/)

```javascript
var removeKdigits = function (num, k) {
  const stk = [];
  for (const ch of num) {
    while (stk.length && k && ch < stk[stk.length - 1]) {
      stk.pop();
      --k;
    }
    if (stk.length || ch !== "0") stk.push(ch);
  }
  while (k--) stk.pop();
  return !stk.length ? "0" : stk.join("");
};
```

- 数字大小主要取决于其左侧（高位）的数字
- 贪心 -> 遍历时优先移除较大的数字
- 单调栈 -> 维护递增序列
- 前导 0 -> `if (stk.length || ch !== "0")`，若栈不为空，`ch` 为 `"0"` 时非前导 0
