```javascript
var permuteUnique = function (nums) {
  nums.sort((a, b) => a - b);
  const ans = [];
  const visited = new Array(nums.length).fill(false);
  const backTracking = (path) => {
    if (path.length === nums.length) {
      ans.push(path.slice());
      return;
    }
    for (let i = 0; i < nums.length; ++i) {
      if (i > 0 && nums[i] === nums[i - 1] && !visited[i - 1]) continue;
      if (visited[i]) continue;
      path.push(nums[i]);
      visited[i] = true;
      backTracking(path);
      path.pop();
      visited[i] = false;
    }
  };
  backTracking([]);
  return ans;
```

当前元素和前一元素相同且前一元素未选择时，说明当前选择为跳过前一元素，直接选择当前元素，导致重复排列，因此通过 `continue` 跳过当前循环
