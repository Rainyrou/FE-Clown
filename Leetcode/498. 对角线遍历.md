[498. 对角线遍历](https://leetcode.cn/problems/diagonal-traverse/description/)

- m 行 n 列矩阵的任意元素 `mat[i][j]` 所在对角线标识为 `k = i + j`，k 的取值范围为 0 到 m + n - 2
- 由 `i = k - j ≥ 0` → `j ≤ k`
- 由 `i = k - j < m` → `j > k - m` → `j ≥ k - m + 1`
- 由 `0 ≤ j < n`
- k 为偶数时，对角线从左下至右上遍历，k 为奇数时，对角线从右上到左下遍历

```javascript
var findDiagonalOrder = function (mat) {
  const m = mat.length,
    n = mat[0].length;
  const ans = [];
  for (let k = 0; k < m + n - 1; ++k) {
    const minJ = Math.max(k - m + 1, 0),
      maxJ = Math.min(k, n - 1);
    if (k % 2 === 0) {
      for (let j = minJ; j <= maxJ; ++j) ans.push(mat[k - j][j]);
    } else {
      for (let j = maxJ; j >= minJ; --j) ans.push(mat[k - j][j]);
    }
  }
  return ans;
};
```
