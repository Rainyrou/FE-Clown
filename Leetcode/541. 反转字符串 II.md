```javascript
const reverseStr = (s, k) => {
  let ans = "";
  for (let i = 0; i < s.length; i += k * 2) {
    const p1 = s
      .slice(i, i + k)
      .split("")
      .reverse()
      .join("");
    const p2 = s.slice(i + k, i + k * 2);
    ans += p1 + p2;
  }
  return ans;
};
```

1. 少于 k 个：`s.slice(i, i + k)` 截取所有剩余字符，`s.slice(i + k, i + k * 2)` 返回空字符串
2. 小于 2k 但大于等于 k 个：同理
