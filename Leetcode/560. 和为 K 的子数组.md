[和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/?envType=study-plan-v2&envId=top-100-liked)

###### 暴力破解 -> 超出时间限制

```javascript
var subarraySum = function(nums, k) {
  let count = 0;
  for(let i = 0;i < nums.length;++i) {
    let curSum = 0;
    for(let j = i;j < nums.length;++j) {
      curSum += nums[j];
      if(curSum === k) ++count;
    }
  }
  return count;
};
```

###### 前缀和 + 哈希表

```javascript
var subarraySum = function(nums, k) {
  const map = new Map();
  map.set(0, 1);
  let prefixSum = 0, ans = 0;
  for(const num of nums) {
    prefixSum += num;
    if(map.has(prefixSum - k)) ans += map.get(prefixSum - k);
    map.set(prefixSum, (map.get(prefixSum) || 0) + 1);
  }
  return ans;
};
```

- `map.set(0, 1)` 初始化以处理从数组起始至当前位置的和为 k 的场景
- 正确逻辑在当前元素前的所有元素中查找，若调换第 28 和 29 行则导致当前的前缀和匹配当前的条件（即将当前状态也置于历史记录中再查找），错误统计不该存在的子数组，在 `k = 0` 的情况下必现
