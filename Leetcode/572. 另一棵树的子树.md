[572. 另一棵树的子树](https://leetcode.cn/problems/subtree-of-another-tree/description/)

```javascript
var isSubtree = function (root, subRoot) {
  if (!root || !subRoot) return false;
  return (
    isSame(root, subRoot) ||
    isSubtree(root.left, subRoot) ||
    isSubtree(root.right, subRoot)
  );
};

const isSame = (node1, node2) => {
  if (!node1 && !node2) return true;
  if (!node1 || !node2 || node1.val !== node2.val) return false;
  return isSame(node1.left, node2.left) && isSame(node1.right, node2.right);
};
```

`isSubtree` 不存在两棵树均为空的场景，由题意有 `root` 节点数 ≥ 1，`subRoot` 节点数 ≥ 1，则初始调用 `isSubtree`时，两棵树非空，而递归调用 `isSubtree` 时，`subRoot` 不变（非空）
