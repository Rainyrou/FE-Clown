[7. 整数反转](https://leetcode.cn/problems/reverse-integer/)

```javascript
var reverse = function (x) {
  let ans = 0;
  while (x) {
    ans = ans * 10 + (x % 10);
    if (ans > Math.pow(2, 31) - 1 || ans < -Math.pow(2, 31)) return 0;
    x = ~~(x / 10); // x = Math.trunc(x / 10);
  }
  return ans;
};
```

1. 无需单独处理符号，因为 JavaScript 中负数取模运算保留负号，拼接后结果自然继承原数字符号
2. 反转本质为提取末位 -> 拼接 -> 移除末位即原数字的最低位 → 反转数字的最高位 + 次低位 → 次高位 + ...
3. `x / 10` → 移除末位 +  `~~` -> 向零取整

向零取整：

```js
x = Math.trunc(x / 10);
const division = x / 10;
x = division >= 0 ? Math.floor(division) : Math.ceil(division);
x = parseInt(x / 10, 10);
```
