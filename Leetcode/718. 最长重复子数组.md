[718. 最长重复子数组](https://leetcode.cn/problems/maximum-length-of-repeated-subarray/description/)

区分 [1143. 最长公共子序列](https://leetcode.cn/problems/longest-common-subsequence/description/)

```javascript
var findLength = function (nums1, nums2) {
  const m = nums1.length,
    n = nums2.length;
  const dp = new Array(m + 1).fill(0).map(() => new Array(n + 1).fill(0));
  let ans = 0;
  for (let i = 1; i <= m; ++i) {
    for (let j = 1; j <= n; ++j) {
      if (nums1[i - 1] === nums2[j - 1])
        dp[i][j] = Math.max(dp[i][j], dp[i - 1][j - 1] + 1);
      ans = Math.max(ans, dp[i][j]);
    }
  }
  return ans;
};
```

若 `nums1[i - 1] !== nums2[j - 1]`，则以当前元素为结尾的公共子数组不存在，`dp[i][j]` 为 0
