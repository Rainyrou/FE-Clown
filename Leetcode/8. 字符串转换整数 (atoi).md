[8. 字符串转换整数 (atoi)](https://leetcode.cn/problems/string-to-integer-atoi/description/)

```javascript
var myAtoi = function (s) {
  const ans = parseInt(s, 10);
  const max = Math.pow(2, 31) - 1,
    min = -Math.pow(2, 31);
  if (isNaN(ans)) return 0;
  else if (ans > max) return max;
  else if (ans < min) return min;
  else return ans;
};
```

题目要求：

1. 无视前导空格
2. 返回有符号整数
3. 无视整数部分后的字符
4. 整数在 32 位有符号整数范围
5. 其他情况返回 0

`parseInt` 转换满足前三点，只需单独处理后两点情况 [parseInt - JavaScript | MDN](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/parseInt)
