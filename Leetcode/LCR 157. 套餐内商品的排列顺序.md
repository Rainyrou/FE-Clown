[LCR 157. 套餐内商品的排列顺序](https://leetcode.cn/problems/zi-fu-chuan-de-pai-lie-lcof/description/)

- `arr[i] === arr[i-1]` -> 判断当前字符和前一字符是否相同 + `!used[i-1]` -> 判断前一相同字符在当前层是否被选择
- 两者同时满足 → 同一层选择相同字符，则产生重复排列 → 跳过
- `used[i-1]` → 前一相同字符在不同层已被选择 → 无需跳过

```javascript
var goodsOrder = function (goods) {
  const arr = goods.split("").sort(),
    ans = [],
    visited = new Array(arr.length).fill(false);
  const backTracking = (path) => {
    if (path.length === arr.length) {
      ans.push(path.join(""));
      return;
    }
    for (let i = 0; i < arr.length; ++i) {
      if (i > 0 && arr[i] === arr[i - 1] && !visited[i - 1]) continue;
      if (visited[i]) continue;
      path.push(arr[i]);
      visited[i] = true;
      backTracking(path);
      path.pop();
      visited[i] = false;
    }
  };
  backTracking([]);
  return ans;
};
```
