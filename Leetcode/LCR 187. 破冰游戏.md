[LCR 187. 破冰游戏](https://leetcode.cn/problems/yuan-quan-zhong-zui-hou-sheng-xia-de-shu-zi-lcof/description/)

- 只有一个人时胜利者对应下标为 0，由此逆推其初始时所在位置
- 每次杀死一个人即将数组向前移动 `target` 位，因为每次均从死者的下一位置开始重新报数

```javascript
var iceBreakingGame = function (num, target) {
  let ans = 0;
  for (let i = 2; i <= num; ++i) ans = (ans + target) % i;
  return ans;
};
```
