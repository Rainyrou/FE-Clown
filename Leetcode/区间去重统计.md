题目描述：输入一个数组，将其划分为若干段，使得各段无重复数字，最少多少段​ 
输入​ ​ 1 2 1 3 2 3 4​​​ ​ 
输出​ ​ 3​​​ ​ 
解释​ [1,2], [1,3,2], [3,4]

```js
var minSegment = function (nums) {
  if (nums.length === 0) return 0;
  let ans = 1;
  const set = new Set();
  for (const num of nums) {
    if (set.has(num)) {
      ++ans;
      set.clear();
    }
    set.add(num);
  }
  return ans;
};
console.log(minSegment([1, 2, 1, 3, 2, 3, 4])); // 3
```
