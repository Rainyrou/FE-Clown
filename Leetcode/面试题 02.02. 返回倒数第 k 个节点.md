[面试题 02.02. 返回倒数第 k 个节点](https://leetcode.cn/problems/kth-node-from-end-of-list-lcci/)

```javascript
var kthToLast = function(head, k) {
  let cur = head, len = 0;
  while(cur) {
    ++len;
    cur = cur.next;
  }
  cur = head;
  for(let i = 0;i < len - k;++i) cur = cur.next;
  return cur.val;
};
```
